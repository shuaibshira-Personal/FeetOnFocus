New Feature: AI Vision Processing

Enable direct PDF/image analysis using Google Gemini Vision AI for more accurate invoice extraction.

  • ✅ No text extraction needed - AI reads PDFs directly
  • 🎯 Better accuracy - Understands visual layout and context
  • 🆓 Free to use - Google Gemini free tier included
  • 🚀 Faster training - One-click supplier learning
Current Processing Method
Uses PDF.js/OCR to extract text, then processes with local Ollama AI
Sends PDF/image directly to Google Gemini Vision for analysis
Google Gemini API Configuration

  1. Go to https://makersuite.google.com/app/apikey
  2. Sign in with your Google account
  3. Click "Create API Key in new project" or "Create API Key"
  4. Copy the generated API key
  5. Paste it in the field above and click "Test Key"
🆓 Free Tier Limits:
• 60 requests per minute
• 1000 requests per day
• Perfect for personal/small business use
